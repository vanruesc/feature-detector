▸ features

A feature map.

Each Feature in the map can be retrieved using its respective FeatureId. Custom Features may also be added to this map.

▸ root

The global scope of the current environment.

This field assumes one of the following values depending on the environment:

  • browser: window
  • worker: self
  • node.js: global
  • unknown: this
